This project contains configuration files and scripts common to the SciJava
software stack.

At the moment, it contains a Maven POM that serves as a base for all
Maven-based SciJava code, including:

  ImageJ1 - http://github.com/fiji/imageja
  ImgLib  - http://github.com/imagej/imglib
  ImageJ2 - http://github.com/imagej/imagej
  SCIFIO  - http://github.com/openmicroscopy/bioformats
  LOCI    - http://github.com/uw-loci/loci-software (coming soon)

This project is also an appropriate place for scripts that are commonly useful
across these various projects.
